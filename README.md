# Concluciones

#### Con los resultados que nos dieron al ejecutar el programa podemos concluir diferentes aspectos sobre los metodos que se crearon para este programa.

## Metodo Insert

Como se puede observar en el output este metodo tiene muchas mas comparaciones que el metodo shell debido a que este metodo tiene que estar comparando cada vez que se "intercambia" el numero para poder insertar el numero en el lugar correcto.

## Metodo Shell

Este metodo es mucho mas eficiente con respecto a las comparaciones que tiene que realizar debido a que este toma saltos(gap) para poder comparar los diferentes datos de forma mas eficiente, pero la desventaja de este metodo es que las iteraciones son mucho mas que las del metodo insert pero aunque sean mas son mucho mas cortas o liviana.
